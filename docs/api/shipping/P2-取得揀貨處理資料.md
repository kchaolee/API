# P2 - 取得揀貨處理資料

## 請求參數

### 內層請求參數（加密前）

requestId			請求編號	String(36)
requestFnName			請求來源	String(30)
requestAction			請求操作	String(30)
wmsOrderNumber			WMS出貨單號	String(10)
storerCode			貨主代碼	String(15)
externOrderNumber			外部出貨編號	String(30)
assignExternReceiptNumber			指定採購單號	String(30)
currentLineNo			目前訂單項次	String(5)


依內層請求參數， 於 \src\SEG.WmsAPI\Models\Requests\OrderRequests.cs 中 建立 P2Request  class 

 
## 回應參數
requestId			請求編號	String(36)
status			API回應結果代碼	String(1)
message			API回應結果訊息	String(255)
data			回覆資料內容	Object
  warehouseCode		倉庫	String(10)
  wmsOrderNumber		WMS出貨單號	String(10)
  storerCode		貨主代碼	String(15)
  externOrderNumber		外部出貨編號	String(30)
  customerName		客戶名稱	String(50)
  assignExternReceiptNumber		指定採購單號	String(30)
  pickingData		待揀訂單項目	Object
    lineNo	WMS項次	String(5)
    externLineNo	外部項次	String(20)
    sku	貨號	String(50)
    descr	品名	String(200)
    packQtyRequired	需求件數	String(10)
    packQtyPicked	已揀件數	String(10)
    pickingUom	揀貨單位	String(18)
    lineRemark	明細備註	String(150)
    totalKG	累計公斤	String(14)
errors			錯誤訊息內容	Array



依回應參數， 於 \src\SEG.WmsAPI\Models\Responses 建立 P2Responses.cs
