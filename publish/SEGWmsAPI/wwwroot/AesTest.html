2. JSON 字串

📋 加密步驟：
  • Minimize JSON（移除多餘空白）
  • 提取 AES Key 的前 32 byte 作為金鑰
  • AES 256 加密（CBC 模式，PKCS7 填充）
  • 轉換為 Base64 並包裝為 {"RequestData": "..."}
🔓 解密步驟：
  • 從 RequestData/ReturnData 提取 Base64 加密資料
  • 提取 AES Key 的前 32 byte 作為金鑰
  • AES 256 解密（CBC 模式，PKCS7 填充）
  • 返回原始 JSON 字串

📖 使用說明

🔒 加密模式：
  1. 輸入 AES Key
  2. 輸入要加密的 JSON 字串
  3. 選擇「加密 Encrypt」
  4. 點擊「執行 Execute」按鈕
  5. 輸出結果會自動包裝為 {"RequestData": "..."} 格式
  6. 點擊「複製 Copy」可複製結果到剪貼簿

🔓 解密模式：
  1. 輸入 AES Key（必須與加密時相同）
  2. 輸入要解密的資料（可以帶 RequestData/ReturnData 包裝或純 Base64）
  3. 選擇「解密 Decrypt」
  4. 點擊「執行 Execute」按鈕
  5. 輸出結果為原始 JSON 字串
