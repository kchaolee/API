# P1 - 取得出貨訂單清單

## 請求參數

### 內層請求參數（加密前）

requestId		請求編號	String(36)
storerCode		貨主代碼	String(15)
docStatus		單據狀態	String(10)
assignExternReceiptNumber		指定採購單號	String(30)

依內層請求參數， 於 \src\SEG.WmsAPI\Models\Requests\OrderRequests.cs 中 建立  P1Request  class 

 
## 回應參數
requestId		請求編號	String(36)
status		API回應結果代碼	String(1)
message		API回應結果訊息	String(255)
data		回覆資料內容	Object
  headerList	表頭清單資料列	Array
  assignExternReceiptNumber	指定採購單號	String(30)
  warehouseCode	倉庫	String(10)
  wmsOrderNumber	WMS出貨單號	String(10)
  storerCode	貨主代碼	String(15)
  externOrderNumber	外部出貨編號	String(30)
  customerName	客戶名稱	String(50)
errors		錯誤訊息內容	Array


依回應參數， 於 \src\SEG.WmsAPI\Models\Responses 建立 P1Responses.cs
