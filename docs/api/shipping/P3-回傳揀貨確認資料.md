# P3 - 回傳揀貨確認資料

## 請求參數

### 內層請求參數（加密前）

requestId			請求編號	String(36)
requestFnName			請求來源	String(30)
requestAction			請求操作	String(30)
wmsOrderNumber			WMS出貨單號	String(10)
storerCode			貨主代碼	String(15)
externOrderNumber			外部出貨編號	String(30)
assignExternReceiptNumber			指定採購單號	String(30)
currentLineNo			目前訂單項次	String(5)
sku			貨號	String(50)
packQtyPicked			已揀件數	String(10)
currentKG			本次公斤數	String(14)
assignExpiryDate			到期日	String(10)


依內層請求參數， 於 \src\SEG.WmsAPI\Models\Requests\OrderRequests.cs 中 建立 P3Request  class 

 
## 回應參數
requestId			請求編號	String(36)
status			API回應結果代碼	String(1)
message			API回應結果訊息	String(255)
data			回覆資料內容	Object
  warehouseCode		倉庫	String(10)
  wmsOrderNumber		WMS出貨單號	String(10)
  storerCode		貨主代碼	String(15)
  externOrderNumber		外部出貨編號	String(30)
  customerName		客戶名稱	String(50)
  assignExternReceiptNumber		指定採購單號	String(30)
  shippingLabel		出貨標籤資料	Object
    lblExternOrderNumber	標籤出貨單號資料	String(40)
    lblCustomerName	標籤客戶名稱資料	String(60)
    lblCaseID	標籤箱號資料	String(20)
errors			錯誤訊息內容	Array



依回應參數， 於 \src\SEG.WmsAPI\Models\Responses 建立 P3Responses.cs
